# Raw Sources

이 디렉토리에 원본 소스 문서를 추가하세요.

## 사용법

1. 웹 아티클, 논문, 보고서 등을 이 폴더에 마크다운 파일로 저장
2. 이미지·PDF 등 첨부파일은 `assets/` 하위 폴더에 저장
3. LLM에게 "새 소스 처리해줘" 또는 파일명을 지정하여 처리 요청

## 규칙

- ⚠️ **이 디렉토리의 파일은 LLM이 절대 수정하지 않습니다**
- 원본 그대로 보존되는 소스 오브 트루스(source of truth)입니다
- LLM은 이 파일을 읽어서 `wiki/` 디렉토리에 요약·분석 페이지를 생성합니다

## 추천 도구

- **Obsidian Web Clipper**: 브라우저 확장 프로그램으로 웹 아티클을 마크다운으로 변환
- 클리핑 후 `Ctrl+Shift+D`로 이미지를 로컬에 다운로드 (옵시디언 설정 필요)
