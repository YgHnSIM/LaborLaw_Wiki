# 대한민국 노동법 위키

이 저장소는 대한민국 노동법 자료를 Obsidian 기반 위키로 정리하기 위한 작업 공간입니다.
원본 자료는 `raw/`에 보존하고, LLM은 이를 바탕으로 `wiki/`에 소스 요약, 개념 정리, 판례·행정해석 분석, 입법사 분석 문서를 생성합니다.

## 구조

```text
raw/              원본 소스 보관 공간. LLM이 수정하지 않습니다.
wiki/sources/     법령, 판례, 행정해석, 입법자료 등 소스 요약
wiki/concepts/    노동법 개념, 조문, 판단기준
wiki/entities/    기관, 법원, 위원회, 단체
wiki/analyses/    쟁점 분석, 판례 비교, 입법 과정과 역사적 맥락
wiki/meta/        템플릿, 방법론, 용어집
```

## 우선 수집할 공식 소스

- 국가법령정보센터: 법령과 시행령·시행규칙
- 대법원·헌법재판소: 판례와 결정례
- 고용노동부: 행정해석, 지침, 보도자료
- 중앙노동위원회: 판정례와 조정·심판 관련 자료
- 국회 의안정보시스템: 법률안, 위원회 검토보고서, 회의록, 본회의 처리 과정

## 운영 원칙

- `raw/`의 원본 파일은 수정하지 않습니다.
- 실체적 주장은 가능한 한 공식 소스에 근거합니다.
- 법령은 기준일, 시행일, 개정 여부를 명시합니다.
- 판례는 사건번호, 선고일, 법원, 핵심 법리를 명시합니다.
- 입법 과정 분석은 입안, 소관 위원회, 법제사법위원회, 본회의, 공포·시행 흐름을 추적합니다.
- 작업 이력은 `wiki/log.md`에 기록합니다.

자세한 작성 규칙은 `AGENTS.md`를 따릅니다.
