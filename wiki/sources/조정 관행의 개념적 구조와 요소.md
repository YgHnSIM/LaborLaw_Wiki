# 조정 관행의 개념적 구조와 요소

## 문서 성격

이현경이 「법철학연구」 제20권 제1호에 발표한 2017년 학술논문이다. 흄과 루이스의 조정 해법 모델을 중심으로 관행의 개념적 구조를 분석한다.

## 핵심 요약

- 조정 관행을 조정 문제라는 상황, 규범적 해법, 행위자 사이의 상호작용으로 구성한다.
- 흄·루이스의 조정 해법 모델, 대항 모델, 마머의 복합 모델을 비교한다.
- 기존 이론이 관행의 역동성과 복잡성을 충분히 포착하지 못한다고 비판한다.
- 조정 균형만으로 관행의 규범성을 완전히 설명하기 어렵다고 본다.
- 관행 동학·관행 생태학·관행 규범학을 포함하는 대안적인 법학적 관행 이론을 제안한다.

## 활용 메모

노동관행 판례의 직접 근거라기보다 반복행위가 공동의 기준으로 형성되는 과정을 설명하는 분석적 배경이다.
